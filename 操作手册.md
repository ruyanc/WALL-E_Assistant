# 🤖 WALL-E 桌面宠物 · 操作手册

> 一只以电影《机器人总动员》主角 **WALL-E（瓦力）** 为原型的桌面宠物（**Windows / macOS**）。
> 采用像素风动画形象，会眨眼、张望、挥手、欢呼、撒娇。
> 帮你管理**待办**、**记事**、**定时提醒**，并用「番茄钟」专注与休息。  
> **桌面端（Windows / macOS）**支持 CloudBase **账号登录**、多设备同步与跨账号任务派发（见 [第十六节](#十六账号同步与任务派发桌面端)）。  
> 界面支持 **简体中文 / English** 切换；中文界面工作台名称为 **瓦力桌面助手**。
> English guide: [USER_GUIDE.md](USER_GUIDE.md)

## 一、功能总览

| 功能 | 说明 |
| --- | --- |
| 🐾 桌面宠物 | 像素风 WALL-E 悬浮桌面，透明背景、可拖动、可缩放、始终置顶 |
| 👀 眼睛底衬 | 镜片区域铺白色底衬，随瓦力缩放同步，不超出眼睛范围 |
| 💡 优先级电灯泡 | 有未完成**个人待办**时，头顶显示红/蓝/绿电灯泡，点击查看任务 |
| 📨 派发徽章 | 已接受「派给我的」在瓦力 **左侧信封**；已接受「我派出的」在 **右侧旗子**；点击直达对应子页 |
| 📋 待办清单 | 卡片式列表、三级优先级下拉、方块勾选完成 |
| 📝 记事本 | 控制台多条目备忘，编辑后自动保存 |
| ⏰ 定时提醒 | 控制台表单添加单次/每天/工作日/每周提醒，到点气泡通知 |
| ⏱️ 番茄钟 | 默认 50 分钟工作 / 10 分钟休息 / 3 轮循环，可自定义 |
| 🔔 休息预警 | 每轮工作结束前 **2 分钟** 气泡提示即将休息 |
| 🛌 休息全屏 | 到点后瓦力放大占满屏幕提醒休息 |
| ⌨️🖱️ 活动联动 | 敲键盘/移鼠标/点鼠标时播放不同动画 |
| 🔔 系统托盘 | 常驻托盘，关闭控制台不退出 |
| 🌐 界面语言 | 控制台可切换简体中文 / English，设置自动保存 |
| ☁️ 账号同步 | **Windows / macOS**：授权码 + 手机号登录，同步待办/记事/提醒/番茄钟设置（CloudBase） |
| 📤 任务派发 | **Windows / macOS**：跨账号派发；接受后在「派给我的」分区处理，**不写入**个人待办 |
| 📦 Windows 安装包 | exe 便携运行；MSI / `install.bat` 安装到用户目录 |
| 🍎 macOS 安装包 | 在 Mac 上运行 `build_mac.sh` 生成 DMG，拖入「应用程序」安装 |

## 二、快速开始

### 方式 A：直接运行（最简单）

1. 双击 `dist\WALL-E.exe`（或安装后的桌面快捷方式）。
2. 瓦力出现在桌面**右下角**，托盘区也有图标。
3. **单击**瓦力或托盘图标 → 打开 **瓦力桌面助手**（英文界面下为 **WALL-E Assistant**）。

### 方式 B：安装到系统（推荐长期使用 · Windows）

1. 先运行 `build.bat` 生成 `dist\WALL-E.exe` 与 `dist\WALL-E.msi`（若尚未打包）。
2. 任选其一安装：
   - 双击 `install.bat silent` 或 `install.bat`：复制 exe 到用户目录；可选 MSI 向导；
   - 或直接双击 `dist\WALL-E.msi` 按向导安装。
3. 双击桌面「**WALL-E Assistant**」快捷方式启动（中文界面标题为 **瓦力桌面助手**）。

### 方式 B2：macOS 安装（DMG）

1. 在 Mac 上执行 `./build_mac.sh`（详见 [BUILD_MAC.md](BUILD_MAC.md)）。
2. 将生成的 `dist/WALL-E.dmg` 发给用户，或在本机双击打开。
3. 把 **WALL-E** 拖入 **Applications（应用程序）** 文件夹。
4. 首次打开若提示未验证开发者：右键 app → **打开**，或在「隐私与安全性」中允许。

> macOS 数据保存在 `~/Library/Application Support/WALL-E`，与 Windows 路径不同；填写相同 **授权码** 并用同一手机号登录后可与 Windows 同步。

### 方式 C：从源码运行（开发者）

```powershell
.\.venv\Scripts\python.exe run.py
```

详见 [第十三节](#十三从源码运行--重新打包)。

## 三、安装与卸载

### 安装

| 步骤 | 操作 |
| --- | --- |
| 1 | 运行 `build.bat` 打包（开发者） |
| 2 | 双击 `install.bat`（可选 MSI 向导），或双击 `dist\WALL-E.msi` |
| 3 | 程序安装到 `%LOCALAPPDATA%\Programs\WALL-E` |
| 4 | 创建桌面快捷方式；快速安装时会附带 `操作手册.md`；可选开机自启 |

**macOS（DMG）**

| 步骤 | 操作 |
| --- | --- |
| 1 | Mac 上运行 `./build_mac.sh` |
| 2 | 打开 `dist/WALL-E.dmg`，将 app 拖入「应用程序」 |
| 3 | 从启动台或应用程序文件夹打开 WALL-E |

### 卸载

双击 `uninstall.bat`：
- 关闭正在运行的瓦力
- 删除安装目录与快捷方式
- 移除开机自启项

> 💡 卸载**不会**删除待办、记事本、提醒与设置（保存在 `%APPDATA%\WALL-E`）。需彻底清除时请手动删除该文件夹。

### 升级已安装版本

```powershell
.\build.bat          # 重新打包 exe + msi
.\install.bat        # 覆盖安装（或手动复制 dist\WALL-E.exe 到安装目录）
```

| 操作 | 效果 |
| --- | --- |

## 五、待办清单与优先级

### 瓦力桌面助手 · 📋 待办

工作台（瓦力桌面助手）默认包含 **待办**、**记事本**、**提醒**、**番茄钟** 四个标签；启用账号同步后另有 **☁️ 账号** 页。

待办页内子标签（启用同步时）：**个人待办** · **派给我的** · **我派出的** · **已完成归档**。

**同步状态**

- 添加任务区域下方显示当前同步状态；登录后可点 **重试同步**。

**添加任务**
- 在输入框填写内容，右侧下拉选择 **高级 / 中级 / 低级**，回车或点「添加」。

**条目样式（卡片布局）**
- 左侧：**方块勾选框** + **优先级色条**（红/蓝/绿）
- 中间：任务文字（长文本自动换行）
- 右下：**优先级下拉**，可随时修改

**完成与删除**
- 点击方块勾选 → 框内打勾、文字删除线并变淡
- **×** 按钮或 **双击** 任务卡片 → 删除（个人待办）
- **已完成归档** 页：个人待办与已完成的派发按完成日期分组；点 **清除已归档** 可清空全部归档；取消勾选可恢复个人待办

**派发分区**（启用同步时）
- **派给我的** / **我派出的** 按状态分区（尚未接受、已接受、已回退/退回、已撤回）。已接受的「派给我的」在瓦力 **左侧信封**、「我派出的」在 **右侧旗子** 显示；点击可直达对应子页。
- 个人待办与派发条目显示 **添加/派出/完成** 时间。

**首次使用示例**

若尚无 `todos.json`，会自动创建三条示例待办：

| 任务 | 优先级 |
| --- | --- |
| 开会 | 高级（红） |
| 交周报 | 中级（蓝） |
| 取快递 | 低级（绿） |

### 头顶电灯泡

有**未完成**待办时，瓦力头部上方显示对应数量的电灯泡：

| 颜色 | 优先级 |
| --- | --- |
| 🔴 红 | 高 |
| 🔵 蓝 | 中（默认） |
| 🟢 绿 | 低 |

点击电灯泡 → 气泡显示：`[高优先级] 任务内容`（等级名称随优先级变化）。

## 六、记事本

控制台 → **📝 记事本**：

- 可添加**多条**小备忘，每条独立文本框。
- 在上方输入框填写后点「添加条目」，或回车添加。
- 编辑内容后**约 0.5 秒自动保存**；也可点「立即保存」。
- 单条右侧 **×** 可删除。
- 数据保存在 `%APPDATA%\WALL-E\notes.json`（旧版 `notes.txt` 会在首次启动时自动迁移）。

## 七、定时提醒

### 控制台添加

打开 **⏰ 提醒** 标签页，填写表单：

| 字段 | 说明 |
| --- | --- |
| **提醒内容** | 如「喝水」「休息」 |
| **触发时间** | 时:分（支持键盘直接输入，无上下箭头） |
| **周期** | 每天 / 工作日 / 每周一至周日 / 单次（指定日期） |
| **日期** | 选「单次」时显示，可点日历选择 |

点 **「添加提醒」** 或内容框回车即可保存；下方列表可查看，选中后点「删除选中」。

**首次使用示例**

若尚无 `reminders.json`，会自动创建：

| 提醒 | 时间 | 周期 |
| --- | --- | --- |
| 喝水 | 10:00 | 每天 |
| 休息 | 22:00 | 每天 |

### 到点行为

- 瓦力**头顶气泡**显示提醒内容。
- 托盘区弹出系统通知。
- 可选提示音（与休息提示音共用开关，在番茄钟页设置）。

## 八、番茄钟与界面语言

控制台 → **⏱️ 番茄钟**：

### 默认参数

| 参数 | 默认值 | 范围 |
| --- | --- | --- |
| 工作时长 | **50** 分钟 | 1–180 |
| 休息时长 | **10** 分钟 | 1–120 |
| 循环次数 | **3** 轮 | 1–12 |
| 休息提示音 | 开 | 开/关 |

即：**工作 50 分钟 → 休息 10 分钟**，循环 **3** 次后结束。

工作/休息/循环时长均支持**键盘直接输入**（已隐藏上下调节按钮）。

### 休息前 2 分钟预警

每轮工作剩余 **2 分钟** 时，瓦力气泡提示「即将开始休息」，并播放说话动画。

### 控制按钮

| 按钮 | 作用 |
| --- | --- |
| ▶ 开始 | 从第 1 轮开始完整循环 |
| ☕ 立即休息 | 马上进入休息 |
| ■ 停止 | 终止计时，回到空闲 |

同页可拖动**瓦力大小**滑块（与桌面拖拽缩放同步）。

### 界面语言

在番茄钟页找到 **「界面语言」**：

| 选项 | 说明 |
| --- | --- |
| 简体中文 | 默认 |
| English | 英文界面 |

切换后即时生效：控制台各标签、托盘菜单、瓦力气泡、休息全屏文案等会一并更新。选择写入 `settings.json` 的 `language` 字段（`zh` / `en`），下次启动自动恢复。

> 你已添加的待办、提醒、记事**内容不会翻译**，仅界面控件与系统提示语切换语言。

## 九、休息全屏提醒

1. **到达休息时间**：瓦力放大占满屏幕，半透明背景 + 倒计时 + 放松小贴士。
2. **休息结束**：自动消失，进入下一轮工作或完成全部循环。
3. **提前结束**：点「我休息好了」或按 **Esc**。
4. **提前开始休息**：控制台 / 右键菜单 / 托盘「立即休息」。

## 十、键鼠活动联动

瓦力会感知你的键鼠活动（**仅 Windows**，全局监测），并切换动画（休息全屏期间暂停）：

## 十一、系统托盘菜单

右键托盘图标：

- 打开瓦力桌面助手
- 显示 / 隐藏宠物
- ▶ 开始番茄钟
- ☕ 立即休息
- ■ 停止计时
- 退出

> 关闭控制台**不会**退出程序。完全退出请用托盘「退出」或瓦力右键菜单「退出」。  
> 退出时会保存本地待办/记事/提醒并立即停止后台同步，**不会**长时间阻塞等待网络（日常编辑约 **4 秒** 后会自动上传）。

## 十二、设置与数据保存位置

| 系统 | 数据目录 |
| --- | --- |
| Windows | `%APPDATA%\WALL-E\` |
| macOS | `~/Library/Application Support/WALL-E/` |

| 文件 | 内容 |
| --- | --- |
| `settings.json` | 番茄钟参数、宠物位置与大小、提示音、`language`、**授权码**（`cloudbase_env_id`）、`sync_paused`（暂停自动同步）等 |
| `todos.json` | 待办清单（含优先级） |
| `notes.json` | 记事本多条目 |
| `reminders.json` | 定时提醒列表 |
| `auth.json` | 登录会话 token（**不含密码**；退出登录会清除） |
| `sync_meta.json` | 上次云同步时间 |
| `sync_config.json` | 可选高级同步配置（备用；优先读 `settings.json` 中的授权码） |
| `assignments.json` | 跨账号派发任务本地缓存 |
| `contact_nicknames.json` | 联系人昵称（派发时可用昵称代替手机号） |

删除这些文件可恢复出厂设置（番茄钟默认恢复为 50/10/3；待办与提醒会重新出现内置示例条目）。

## 十三、从源码运行 / 重新打包

### 环境要求

| 平台 | 要求 |
| --- | --- |
| Windows | Windows 10/11 |
| macOS | macOS 11+（打包 DMG） |
| 通用 | Python 3.10+（项目用 3.12 验证） |

### 首次准备

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements-dev.txt
```

### 直接运行

```powershell
.\.venv\Scripts\python.exe run.py
```

### Windows 一键打包 + 安装

```powershell
.\build.bat       # 生成 dist\WALL-E.exe 与 dist\WALL-E.msi
.\install.bat     # 快速安装或 MSI 向导；复制操作手册
```

仅重新生成 MSI（需已有 exe）：

```powershell
.\build_msi.bat
```

或手动打包：

```powershell
.\.venv\Scripts\python.exe make_icon.py
.\.venv\Scripts\python.exe -m PyInstaller --noconfirm --clean WALL-E.spec
.\.venv\Scripts\python.exe scripts\build_msi.py
```

**Windows 产物：**
- `dist\WALL-E.exe`（单文件便携版，含动画资源）
- `dist\WALL-E.msi`（安装包，可选 WiX 或 Python 回退构建）

**验证脚本（开发者）：**

```powershell
.\.venv\Scripts\python.exe scripts/verify_sync_login_dispatch.py
```

全部单元测试通过后再打包。联机探测：`--live`（需有效登录会话，见 `GUIDE/sync/CLOUDBASE_SETUP.md`）。

### macOS 打包（DMG）

在 Mac 上执行：

```bash
chmod +x build_mac.sh scripts/build_dmg.sh
./build_mac.sh
```

产物：`dist/WALL-E.app`、`dist/WALL-E.dmg`。详见 [BUILD_MAC.md](BUILD_MAC.md)。

### 构建缓存与版本库

项目用 [`.gitignore`](.gitignore) 忽略中间产物，仅追踪复现项目所需的源码、配置，以及 `dist/` 下的**正式安装包**（`WALL-E.exe`、`WALL-E.msi`、`WALL-E.dmg`）。

一键清理本地缓存（**保留** `dist` 安装包）：

```bat
clean_build_artifacts.bat
```

`mobile/.buildozer/` 为 Android 打包缓存，可完全删除；Windows 下若删不干净，可在 WSL 执行 `rm -rf .../mobile/.buildozer`。详见 [GUIDE/BUILD_ARTIFACTS.md](GUIDE/BUILD_ARTIFACTS.md)。

### 项目结构

```
WALL-E/
├─ run.py                 启动入口
├─ build.bat / build_msi.bat / install.bat / uninstall.bat   Windows
├─ clean_build_artifacts.bat   一键清理构建缓存（保留 dist 安装包）
├─ build_mac.sh / BUILD_MAC.md / WALL-E-mac.spec             macOS
├─ installer/WALL-E.wxs   WiX 安装包定义（可选）
├─ scripts/build_msi.py   MSI 回退构建脚本
├─ scripts/build_dmg.sh     DMG 封装脚本
```

## 十四、常见问题 FAQ

**Q：双击 exe 没反应？**
A：看屏幕**右下角**和**托盘区**。右键托盘 →「显示/隐藏宠物」。

**Q：SmartScreen 拦截？**
A：点「更多信息 → 仍要运行」。本地宠物与待办可离线使用；**账号同步与任务派发**需联网访问 CloudBase。

**Q：番茄钟不是 50/10/3？**
A：检查 `%APPDATA%\WALL-E\settings.json`，或删除该文件恢复默认。

**Q：怎么开机自启？**
A：`install.bat` 时选 `Y`；或把快捷方式放入 `shell:startup`。

**Q：换电脑保留数据？**
A：未登录时复制整个 `%APPDATA%\WALL-E`。已启用同步时，在新电脑填写相同授权码并用同一手机号登录即可。

**Q：登录没反应或退出后无法换账号？**
A：先 **退出登录** 再登另一账号；确保已保存授权码。仍失败则完全退出瓦力后重启（请使用最新安装包）。

**Q：派发提示「未找到该手机号用户」？**
A：接收方须至少用 WALL-E **成功登录一次**；双方须使用 **相同授权码**。

**Q：升级后功能没变？**
A：需重新 `build.bat` 并 `install.bat` 覆盖安装，然后重启瓦力。

**Q：退出时程序未响应？**  
A：旧版退出时会阻塞等待云同步。请升级至最新安装包；新版本退出会立即保存本地数据并快速结束进程。

**Q：待办/提醒示例想恢复默认？**
A：删除 `%APPDATA%\WALL-E\todos.json` 或 `reminders.json` 后重启（会重新写入示例数据）。

**Q：exe 和 msi 有什么区别？**
A：exe 可直接运行；msi 适合双击安装到用户目录。二者功能相同。`install.bat` 在检测到 msi 时可选择 MSI 向导安装。

**Q：怎么切换英文界面？**
A：打开瓦力桌面助手 → **番茄钟** → **界面语言** → 选 English。或在 `settings.json` 中将 `language` 设为 `"en"`。

**Q：Mac 版怎么安装？**
A：使用 `dist/WALL-E.dmg`，拖入「应用程序」。需自行在 Mac 上运行 `build_mac.sh` 生成（Windows 无法打 DMG）。

**Q：Mac 和 Windows 数据能同步吗？**
A：可以。macOS 与 Windows 桌面版均支持 CloudBase：填写相同 **授权码** 并用同一手机号登录，即可同步待办/记事/提醒/设置；详见 [第十六节](#十六账号同步与任务派发桌面端) 与 `GUIDE/sync/CLOUDBASE_SETUP.md`。

## 十六、账号同步与任务派发（桌面端）

> 管理员部署 CloudBase 见 **[GUIDE/sync/CLOUDBASE_SETUP.md](GUIDE/sync/CLOUDBASE_SETUP.md)**。  
> 方案说明见 `GUIDE/ToDoList/跨平台账号同步方案.md`、`GUIDE/ToDoList/跨账号任务派发方案.md`。

### 工作台标题

| 状态 | 中文界面 | 英文界面 |
| --- | --- | --- |
| 未登录 | **瓦力桌面助手** | **WALL-E Assistant** |
| 已登录 | **`{手机号}的瓦力桌面助手`** | **`{phone}'s WALL-E Assistant`** |

### 首次配置

1. 打开瓦力桌面助手 → **☁️ 账号**。
2. 向管理员获取 **授权码**（CloudBase 环境 ID），填入并 **保存授权码**。
3. 选择登录方式：
   - **密码登录**：手机号 + 密码；
   - **验证码登录**：发送验证码 → 输入验证码；
   - **新用户注册**：手机号 + 密码 + 注册验证码。
4. 点击 **登录并同步**。登录成功后自动从云端拉取数据。

手机号会自动规范为 `+86 138xxxxxxxx` 格式。

### 多设备同账号

多台 Windows / macOS（或 Android）填写 **相同授权码**，用 **同一手机号** 登录。待办、记事、提醒、番茄钟相关设置按 **最后写入优先（LWW）** 合并。本地修改约 **4 秒** 后上传，并每 **15 分钟** 定时同步。待办页或账号页可 **立即同步 / 重试同步**；账号页可 **暂停自动同步**（手动同步仍可用）。

### 跨账号任务派发

1. 账号 A、B 均保存授权码并 **各登录至少一次**。
2. A → 待办 → **我派出的**：填写 B 的手机号（或联系人昵称）、任务标题与可选 **任务说明** → **派发任务**。
3. B → **派给我的**（按状态分区：尚未接受 / 已经接受 / 已经回退 / 已撤回）：**接受** / **拒绝**（须填理由）；已接受可 **完成**。
4. A → **我派出的**（分区：对方尚未接收 / 对方已接受 / 对方退回 / 已撤回）：可对未完成任务 **撤回**（须填理由）。状态变更时瓦力会弹出气泡通知；点击瓦力 **信封 / 旗子** 可直达对应子页。

接受后的协作任务留在 **派给我的** 分区处理，**不会**写入个人待办。已完成个人待办与已完成派发任务统一在 **已完成归档** 页按完成日期分组；仅在该页 **清除已归档**。

### 联系人与昵称

在 **账号** 页 **联系人** 区可为常用手机号设置昵称，派发时可直接输入昵称。昵称仅保存在本机 `contact_nicknames.json`；**切换账号或退出登录时会清空**。

### 退出登录与切换账号

- 点击 **退出登录** 会清除本地会话与业务缓存，恢复为未登录状态。
- **建议**在同一台共用电脑上先退出再登录另一账号。
- 若直接登录另一账号，程序会自动识别 `user_id` 变化并清空本地数据后再同步，避免看到上一账号内容。

### 常见问题（同步）

| 现象 | 处理 |
| --- | --- |
| 请先填写授权码 | 在账号页保存授权码 |
| 登录中长时间无响应 | 检查网络；新版本登录在后台进行，成功后标题变为「xxx的瓦力桌面助手」 |
| 退出后无法登录另一账号 | 升级至最新安装包；完全退出程序后重试 |
| 未找到该手机号用户 | 让对方先登录一次 WALL-E |
| 派发失败 | 查看弹窗错误；确认授权码一致、对方已登录 |
| token 过期 | 在账号页重新登录 |
| 退出时未响应 | 升级至最新安装包（旧版退出会阻塞等待同步） |
