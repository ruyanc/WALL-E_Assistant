# 🤖 WALL-E 手机版 · 操作手册

> 以电影《机器人总动员》主角 **WALL-E（瓦力）** 为原型的 **Android 伴侣应用**。
> 采用与桌面版相同的像素风动画资源，在手机上陪你专注、休息，并管理待办。
>
> 本应用为独立 **Kivy** 项目，与 Windows 桌面版（PySide6）**不是同一个安装包**，功能范围也更精简。

## 一、功能总览

| 功能 | 说明 |
| --- | --- |
| 🐾 瓦力动画 | idle 待机、张望、说话、开心等像素动画 |
| 📋 待办清单 | 增删任务、方块勾选完成、三级优先级（高级 / 中级 / 低级） |
| ⏱️ 番茄钟 | 50 分钟专注 / 10 分钟休息，大屏倒计时 |
| 📱 竖屏应用 | 底部三页切换：瓦力 · 待办 · 番茄钟 |

**当前版本尚未包含：** 定时提醒、记事本、桌面悬浮、系统托盘、键鼠联动、休息全屏遮罩、与 Windows 版数据自动同步。

## 二、与桌面版的区别

| 项目 | 手机版（Android） | 桌面版（Windows / macOS） |
| --- | --- | --- |
| 运行环境 | APK，Android 7.0+（API 24+） | exe / msi（Win）；dmg / app（Mac） |
| 界面语言 | 仅中文 | 简体中文 / English 可切换 |
| 瓦力展示 | 应用内全屏区域播放动画 | 桌面悬浮窗，可拖动、缩放、置顶 |
| 待办 | 卡片列表 + 色条 + 勾选 / 删除 | 控制台卡片 + 头顶电灯泡快捷入口 |
| 番茄钟 | 固定 50 / 10 分钟，手动开始 | 可自定义时长与轮数，带休息预警与全屏 |
| 提醒 / 记事 | ❌ | ✅ |
| 数据位置 | 应用私有目录 | Win：`%APPDATA%\WALL-E`；Mac：`~/Library/Application Support/WALL-E` |
| 数据互通 | ❌ 不自动同步 | — |

若你同时使用两个版本，待办需要分别在各自应用里维护，或自行复制 `todos.json`（格式兼容，但路径不同，见下文）。

## 三、安装到手机

### 方式 A：安装调试 APK（自行打包或他人分发）

1. 取得 APK 文件（常见文件名示例：`walle-1.0.0-arm64-v8a-debug.apk`）。
2. 将 APK 复制到手机（数据线、网盘、微信文件助手等均可）。
3. 在手机**文件管理器**中点击该 APK。
4. 若系统提示「禁止安装未知应用」，到 **设置 → 安全 / 应用安装** 中，为当前文件管理器或浏览器**允许安装未知来源**。
5. 按提示完成安装，桌面会出现 **WALL-E** 图标。

> **架构说明：** 打包脚本可能生成 `arm64-v8a`（多数近年手机）或 `armeabi-v7a`（较老 32 位机）版本。装错架构会提示无法安装，请换对应 APK 重试。

### 方式 B：通过电脑 adb 安装

手机开启**开发者选项 → USB 调试**，连接电脑后执行：

```bash
adb install path/to/walle-1.0.0-arm64-v8a-debug.apk
```

### 卸载

与卸载普通应用相同：**设置 → 应用 → WALL-E → 卸载**。  
卸载后应用私有目录内的待办数据会一并清除。

## 四、界面与导航

打开应用后，主界面分为上下两部分：

- **上方**：当前页面内容（瓦力 / 待办 / 番茄钟）。
- **下方**：三个导航按钮 — **瓦力** · **待办** · **番茄钟**，点击即可切换。

应用切换到后台再返回时，计时会继续（Android 会尽量保持进程）；若系统杀掉了应用，番茄钟进度会丢失，需重新开始。

---

## 五、瓦力页

瓦力页中央显示像素风 WALL-E 动画，默认循环**待机（idle）**。

下方有三个快捷按钮，可临时切换动画：

| 按钮 | 效果 |
| --- | --- |
| 张望 | 播放张望动画 |
| 说话 | 播放说话动画 |
| 开心 | 播放开心动画 |

动画播放结束后会自动回到待机。此处为手动预览，**不会像桌面版那样根据键鼠活动自动切换**。

## 六、待办清单

### 添加任务

1. 切到 **待办** 页。
2. 在顶部输入框填写任务内容。
3. 右侧下拉选择优先级：**高级** / **中级** / **低级**（默认中级）。
4. 点 **添加**，或键盘上按**完成 / 回车**。

### 列表说明

- 左侧**彩色竖条**表示优先级：红 = 高级，蓝 = 中级，绿 = 低级。
- **未完成**任务排在前面，同优先级按创建时间排序。
- **已完成**任务排在列表后方，文字变灰并带删除线。

### 完成与删除

- 点任务右侧 **□** → 标记完成（变为 **✓**）；再点一次可取消完成。
- 点 **×** → 永久删除该任务。

### 首次使用示例

第一次安装、尚无本地数据时，会自动创建三条示例待办，方便熟悉界面：

- 开会（高级）
- 交周报（中级）
- 取快递（低级）

可直接勾选完成或删除，也可继续添加自己的任务。

## 七、番茄钟

切到 **番茄钟** 页，中央为大号倒计时 **分:秒**。

| 按钮 | 作用 |
| --- | --- |
| **开始 50 分钟** | 进入专注倒计时（状态显示「专注中…」） |
| **休息 10 分钟** | 进入休息倒计时（状态显示「休息中 ☕」） |
| **停止** | 结束当前计时，状态变为「已停止」 |

倒计时归零后显示 **「时间到！」**，计时自动停止。

**与桌面版的差异：**

- 时长固定为 **50 / 10 分钟**，暂不能在应用内修改。
- 无「每轮结束前 2 分钟预警」、无全屏休息遮罩、无与瓦力联动的气泡通知。
- 建议到点后自行切到 **瓦力** 页点「开心」庆祝一下休息 😊

## 八、数据保存与备份

### 保存内容

目前仅 **待办清单** 会持久化保存；番茄钟无历史记录。

### 保存位置

| 环境 | 路径 |
| --- | --- |
| Android 真机 | 应用私有目录下的 `todos.json`（用户无法直接浏览，卸载即删） |
| 电脑调试运行 | 用户目录 `~/.walle-mobile/todos.json`，或通过环境变量 `WALLE_MOBILE_DATA` 指定目录 |

JSON 结构与桌面版 `todos.json` **字段兼容**（含 `next_id`、`tasks`、优先级等），但**两个版本不会自动同步**。

### 备份建议

- 重要待办请定期在应用内自行核对，或请熟悉 adb 的用户从私有目录导出 `todos.json`。
- 换机前若无导出，卸载旧机应用后数据无法恢复。

## 九、常见问题 FAQ

**Q：和电脑上的 WALL-E 是同一个软件吗？**  
A：同源项目、同一套瓦力素材，但手机版是单独的 Android 应用，功能更少，数据也不互通。

**Q：安装时提示「应用未安装」或「与现有软件冲突」？**  
A：检查 APK 是否损坏、架构是否匹配（arm64 / armeabi-v7a）；若曾安装过同包名不同签名版本，需先卸载旧版再装。

**Q：允许「未知来源」安全吗？**  
A：调试 APK 仅供测试。只安装你信任的来源；上架应用商店需使用正式签名的 release 包。

**Q：待办会同步到电脑吗？**  
A：不会。需手动维护两份，或自行复制 JSON 文件（注意路径与权限）。

**Q：番茄钟后台还在走吗？**  
A：短时间内切到其他应用通常继续；若系统回收内存杀死进程，计时会中断。

**Q：为什么没有提醒、记事、桌面宠物？**  
A：Android 系统限制与开发优先级原因，首版仅实现核心三项；后续版本可能逐步补齐。

**Q：通知权限有什么用？**  
A：安装包已声明通知相关权限，为后续「到点提醒」等功能预留；**当前版本尚未使用系统通知**。

## 十、开发者：打包与调试

普通用户只需阅读上文第三节安装 APK 即可。若你需要**自己编译 APK** 或在 Windows 上**预览界面**，请参阅同目录下的 **[BUILD_ANDROID.md](./BUILD_ANDROID.md)**，内容包括：

- Windows 本地 Kivy 预览（`prepare_assets.py` + `main.py`）
- WSL2 + Buildozer 打包流程
- 虚拟环境与常见报错处理
- 调试 APK 输出路径：`mobile/bin/*.apk`

简要预览命令（在项目根目录，需已安装 Python 与 Kivy）：

```powershell
cd mobile
..\.venv\Scripts\python.exe prepare_assets.py
..\.venv\Scripts\python.exe main.py
```
